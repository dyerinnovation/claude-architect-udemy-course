## Description

The course Claude Certified Architect: Anthropic Exam Prep covers the essential lessons of designing, orchestrating, and operating production Claude-based agent systems at the depth tested on Anthropic's Claude Certified Architect exam (v0.1). It is built for solution architects and senior engineers with working LLM API experience who want domain-by-domain coverage of the five exam areas plus the six scenarios that drive the question set, not a survey of generative AI.

Instruction maps directly to the official exam guide weights: Agentic Architecture & Orchestration (27%), Tool Design & MCP Integration (18%), Claude Code Configuration & Workflows (20%), Prompt Engineering & Structured Output (20%), and Context Management & Reliability (15%). Each domain is taught through the concrete decisions the exam asks about — programmatic enforcement vs prompt guidance, `tool_choice` selection, CLAUDE.md hierarchy, Message Batches API tradeoffs, escalation triggers, claim-source provenance — and reinforced with scenario labs and a 40-question scored practice exam.

**Skills you'll learn:**
- Agentic Loop Control with `stop_reason`
- Hub-and-Spoke Multi-Agent Orchestration
- Tool Description & Selection Design
- MCP Server Configuration and Error Response Design
- Claude Code Workflow Configuration (CLAUDE.md, skills, slash commands)
- Structured Output via Tool Use and JSON Schemas
- Message Batches API Decisioning
- Context Preservation and Escalation Patterns

**What you'll build:**
- A Customer Support Resolution Agent with programmatic prerequisite gates and a `PostToolUse` normalization hook
- A Multi-Agent Research Pipeline with parallel subagent spawning and claim-source mappings
- A Team Workflow Configuration covering project CLAUDE.md, `.claude/rules/` glob scoping, and a forked-context skill
- A Claude Code CI/CD Pipeline Integration using `-p` with `--output-format json --json-schema`
- A Structured Data Extraction Pipeline using `tool_choice: "any"`, validation-retry loops, and batch processing by `custom_id`
- A CI Code Review prompt system with explicit criteria, few-shot examples, and multi-pass per-file plus cross-file review

**Scope:** ~18.5 hours of instruction and labs, 10 sections, 40-question scored practice exam, Intermediate level.
