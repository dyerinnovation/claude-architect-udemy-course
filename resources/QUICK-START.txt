CRITICAL CONCEPTS (Must Know Cold)

1. stop_reason = "tool_use" → execute tool + loop
2. stop_reason = "end_turn" → return to user + stop
3. Domain 1 = 27% (agentic architecture, tool loops)
4. Escalate on: policy gap, explicit request, or stuck
5. Do NOT escalate on: frustration, length, confidence
6. tool_choice = "auto": model chooses
7. tool_choice = "any": model MUST call a tool
8. tool_choice = {"type":"tool","name":"X"}: must call THIS tool
9. Error fields: errorCategory + isRetryable + description
10. Batch API: 50% savings, up to 24hr, no multi-turn tools
11. Sync API: real-time, for blocking workflows
12. .claude/rules/ hierarchy: path-specific > shared > personal
13. Intervention Level 1: explicit criteria (fix vague prompts)
14. Intervention Level 4: architectural redesign
15. GLOB: find files by name/pattern
16. GREP: search file contents
17. READ: load a known file
18. WRITE: new file or complete replacement
19. EDIT: targeted modifications to existing file
20. Master these and you can answer 70%+ of exam questions
