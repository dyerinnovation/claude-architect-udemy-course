# Project Setup Pattern

This project follows the standard Claude Code project structure:

- `CLAUDE.md` at root — project bootstrap, read first
- `.claude/rules/` — persistent conventions and guidelines
- `docs/` — human-facing documentation (monetization, creation guides, naming)
- `claude_projects/` — per-feature project plans and progress tracking

When starting a new workstream, create a tracking entry in `claude_projects/`.
