# Section 2: Claude API Fundamentals Bootcamp

## Overview
- **Domain**: API foundations (prerequisite bootcamp — not an exam domain)
- **Exam Weight**: N/A (foundational)
- **Lecture Count**: 11 lectures
- **Estimated Duration**: ~90 minutes

## Learning Objectives

By the end of this section, students will be able to:
- Construct Messages API requests with correctly placed system prompts, user/assistant turns, and sampling parameters.
- Apply temperature, top_p, top_k, prefilled assistant messages, and stop sequences to shape Claude's output deterministically.
- Implement response streaming and distinguish the SSE event types in their correct order.
- Design XML-tagged prompts and choose between tool_use and JSON-mode strategies for structured output.
- Execute the complete tool use loop end-to-end, including multimodal image inputs in the Messages API.
